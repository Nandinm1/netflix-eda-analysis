## 📌 Project Overview
This project analyzes the Netflix Movies & TV Shows dataset using Python to uncover trends in content, countries, genres, ratings, and release history through exploratory data analysis and visualizations.

## 🎯 Objectives
- Load and inspect the dataset
- Clean missing values and duplicates
- Perform EDA using Pandas
- Build 6+ visualizations
- Generate business insights

## 🛠️ Tech Stack
