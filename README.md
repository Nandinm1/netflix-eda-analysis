## 📌 Project Overview
This project builds and compares three machine learning models to classify Iris flower species using the Scikit-learn library.

## 🎯 Objectives
- Preprocess the dataset
- Engineer useful features
- Train three ML models
- Compare evaluation metrics
- Select the best model

## 🤖 Models
- Logistic Regression
- Random Forest
- K-Nearest Neighbors (KNN)

## 📏 Evaluation
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

> **Tip:** Add screenshots of the confusion matrix and comparison chart in a `screenshots/` folder.
