# Netflix Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies & TV Shows dataset using Python. The objective is to understand the structure of the data, clean missing values, discover patterns, create visualizations, and generate business insights.

## Dataset
- **Dataset:** Netflix Movies & TV Shows
- **Source:** Kaggle
- **File Used:** `netflix_titles.csv`

## Objectives
- Load and inspect the dataset
- Clean missing and duplicate data
- Perform exploratory data analysis (EDA)
- Create multiple visualizations
- Generate data-driven business insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Exploratory Data Analysis
1. Movies vs TV Shows
2. Top content-producing countries
3. Most common ratings
4. Content added by year
5. Most popular genres

## Visualizations
- Bar Chart
- Line Chart
- Histogram
- Scatter Plot
- Pie Chart
- Heatmap
